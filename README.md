# ChatGateAI

Минималистичное Flutter-приложение с авторизацией через Google и встроенным окном ChatGPT.
Проект создан с прицелом на низкое потребление оперативной памяти и простую архитектуру.

## Что делает приложение

1. Пользователь заходит в приложение.
2. Авторизуется через Google.
3. После успешного входа открывается встроенное окно с ChatGPT, чтобы сразу продолжать свои чаты.

Проще говоря: меньше переключений между приложениями и вкладками, больше концентрации на деле.

## Технологии

- **Flutter**
- **google_sign_in**
- **flutter_inappwebview**

## Установка и запуск

### clone repository
```bash
git clone git@github.com:KsArt-IT/ChatGateAI-Flutter.git
cd ChatGateAI-Flutter
```
### run
```bash
flutter pub get
flutter run
```
